# 《判官》3D动画电影分镜脚本

本仓库包含《判官》3D动画电影的前两集分镜脚本。

## 文件说明

| 文件 | 说明 | 分镜数 |
|------|------|--------|
| 判官第一集.xlsx | 第一集完整分镜脚本 | 37 镜 |
| 判官第二集.xlsx | 第二集完整分镜脚本 | 22 镜 |

## 分镜字段说明

每个分镜包含以下核心字段：

- **集数** - 所属集数
- **镜号** - 镜头编号
- **分镜段** - 场景段落名称
- **时间轴** - 镜头起止时间
- **场景** - 内景/外景、日/夜
- **景别** - 特写/近景/中景/全景等
- **人物** - 出场角色
- **动作描述** - 角色动作与情节
- **主画面描述** - 构图、视角、光感详细描述
- **运镜** - 镜头运动方式
- **台词** - 角色对白
- **音效** - 声音设计
- **画面生成提示词** - 用于 AI 生成的提示词
